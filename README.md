# Digital Clock using JavaScript

* Project Description
- A simple Digital Clock application developed using HTML, CSS, and JavaScript
- Displays the current time, date, and day in real time
- JavaScript is used for clock logic, while HTML and CSS handle structure and styling

* Technologies Used
- HTML5 for structure
- CSS3 for styling
- JavaScript for date and time logic

* Features
- Displays hours, minutes, and seconds
- Shows day, date, month, and year
- Updates automatically every second
- Responsive and lightweight design

* How It Works
- Uses the JavaScript Date object to retrieve current date and time
- Uses setInterval to update the clock every second
- Updates the displayed values dynamically using DOM manipulation

* Output
- Real-time display of current time and date
- Continuous updates without page refresh

* Learning Outcomes
- Understanding the JavaScript Date object
- Using setInterval for real-time updates
- DOM manipulation with JavaScript
- Integrating HTML, CSS, and JavaScript
